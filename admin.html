Вход

Как получить токен (один раз):
1. Открыть github.com → Settings → Developer settings → Fine-grained tokens
2. «Generate new token» → Repository access: выбрать только этот репозиторий
3. Permissions → Contents → Read and write
4. Скопировать токен и вставить выше
